# Slideable-calendar
&ensp;This project aims to provide a swipeable schedule note-taking software for recording tasks and to-dos completed in each time period of the day. It features parallel scheduling capabilities, supports time accuracy down to the minute, cross-date scheduling, and allows users to add notes by name.  

&emsp;本项目旨在提供一款可滑动操作的日程安排笔记软件，用于记录一天中各个时间段完成的任务和待办事项。它具备并行日程安排功能，支持精确到分钟的时间设置、跨日期日程安排，并允许用户按名称添加笔记。

&ensp;Project Background:   
&ensp;Due to my lack of planning in my work, colleagues often put aside their current tasks to address new requirements, resulting in none of the tasks being completed. I hope to have a draggable timeline for scheduling events. Although this feature already exists in mobile calendars, I don't want to constantly adjust things on my phone. I want to implement this feature on my computer, allowing me to schedule my tasks into a daily timeline.  
  
The following features are expected to be included:
&ensp;1. 24-hour horizontal timeline display. Double-click to create a new event. The start and end times of each event can be changed by dragging (and the start/end times are displayed in real-time).  
&ensp;2. Support for multiple events running concurrently within a specific time period, displayed in multiple rows.  
&ensp;3. Support for event memory. Historical event records can be viewed each time the event is opened.  
&ensp;4. Support for event export in JSON/PDF/Markdown formats.  
&ensp;5. Rounded corner design and attention to color scheme.  

项目背景：  
&emsp;由于本人工作特别没有规划，导致有同事新的需求总是放下手里的工作去做新的需求，最后导致所有需求都没有完成，希望能有一个能拖动时间轴的日程便签，尽管这个功能手机日历早已实现，但不希望总是用手机调整，希望在电脑上实现这个功能，将自己的代办排进每日的时间轴日程里面。  
预期包含以下功能：  
&emsp;1.每天24H按照横向时间轴显示，双击新建日程，每个日程的开始和结束时间都通过拖动改变（并且实时显示拖动时的开始/结束时间）  
&emsp;2.支持某时间段内多日程并行，并行日程多行显示；  
&emsp;3.支持日程记忆，每次打开能看到历史的日程记录；  
&emsp;4.支持日程导出，导出为json/PDF/Markdown格式；  
&emsp;5.圆角设计，关注配色方案；
